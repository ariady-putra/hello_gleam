## Deployment

Deploying on [Fly.io](https://fly.io):

### Install

```sh
curl -L https://fly.io/install.sh | sh
```

### Deploy

```sh
flyctl launch
```

To deploy future versions of the application run `flyctl deploy` after saving any changes to the source code.
